# Artha AI: Indian Stock Market Insights

Artha AI is a sophisticated AI-powered stock tracker tailored for the Indian market. It provides real-time data for NSE and BSE stocks, combined with advanced sentiment analysis and price predictions powered by Google Gemini.

## Features

- **Real-time Tracking**: Live price updates and interactive charts for all NSE/BSE stocks.
- **AI-Powered Insights**: Deep sentiment analysis and 1-month price targets using Gemini 1.5 Flash.
- **Smart Watchlist**: Manage your favorite stocks and get quick snapshots of market performance.
- **Market Sentiment**: Understand the broader market trend at a glance.
